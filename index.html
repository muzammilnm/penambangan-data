    Tugas 1¶

    Menggambarkan sebuah data dengan 4 kolom dan menampilkan  grafik dan statistik dari data¶

    langkah 1 :

    buat sebuah data menggunakan tools microsoft office exel dan isikan data dengan ketentuan terdapat 4 kolom, lalu simpan dengan extensi '.csv'. untuk tampilan data nya seperti gambar dibawah ini.

    langkah 2 :

    buat file python dan simpan didalam direktori yang sama dengan file exel, dengan catatan file python yang kita buat harus dibuat dengan menggunakan jupyter notebook

    langkah 3 :

    tuliskankan kode program seperti di bawah ini :

    from scipy import stats
import  pandas as pd
pd.read_csv("D_pertemuan1.csv")

    kode diatas untuk memanggil library scipy dan pandas serta membaca file exel yang bernama 'D_pertemuan1.csv'.

    sehingga menghasilkan output :

    menampilkan statistik dari data dengan cara :

    df=pd.read_csv("D_pertemuan1.csv")
for col in df.columns:
    print("===",col,"===")
    print("jumlah sel data keseluruhan =",df[col].count())
    print("mean =",df[col].mean())
    print("median =",df[col].median())
    print("modus =",df[col].mode())
    print("nilai minimal ",df[col].min())
    print("Nilai Max =",df[col].max())
    print("Nilai Varian =",df[col].var())
    print("Kuartil awal Q1 =",df[col].quantile(0.25))
    print("Kuartil tengah Q2 =",df[col].quantile(0.5))
    print("kuartil tengah Q3 =",df[col].quantile(0.75))
    print("skewwnes =","{0:.2f}".format(round(df[col].skew(),2)))
    print("skewwnes =" ,"{0:.6f}".format(round(df[col].skew(),6)))
    print("Standar Deviasi =",df[col].std())
    print("Standar Deviasi =","{0:.2f}".format(round(df[col].std(),2)))
    print("==========================\n")

    kode diatas memanggil beberapa fungsi seperti mean, min, quantile, max. untuk mengetahui nilai-nilai dari data maka harus menampilkan dengan fungsi tersebut.
